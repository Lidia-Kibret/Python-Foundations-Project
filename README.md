# 🧮 Python Foundations Project

This repository contains Python programs developed for the **Python Foundations Daily Assignment**.  
It demonstrates basic Python concepts including loops, functions, conditionals, and user interaction.

---

## 📂 Projects Included

### 1️⃣ Prime Number Checker
Checks whether a number is prime or not.  
**Features:**
- Identifies prime and composite numbers
- Handles special cases (0, 1, negative numbers)
- Provides clear output messages to the user  
**File:** `prime.py`

---

### 2️⃣ Fibonacci Sequence Generator
Generates the Fibonacci sequence for a given number of terms.  
**Features:**
- Starts the sequence with 0 and 1
- Generates any number of terms as input by the user
- Uses efficient looping and variable swapping  
**File:** `fibonacci_sequence.py`

---

### 3️⃣ Tic-Tac-Toe Game
A CLI-based two-player Tic-Tac-Toe game.  
**Features:**
- Two players can play interactively (X and O)
- Checks for wins across rows, columns, and diagonals
- Detects draw situations
- Prevents overwriting an already filled cell  
**File:** `tictactoe.py`

---
